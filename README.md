# uriencode

Tiny python tool that will uri encode a string one or more times

## Usage

    $ python3 uriencode.py
    Enter string to encode: <this is my text>
    Encode how many times? 1
    ** ** ** ** **
    ** encoding **
    ** ** ** ** **
    Encoded string: 
    %3Cthis%20is%20my%20text%3E
